+ 事件类型
    + bindtap 轻轻敲击就会触发这个元素事件
    + bindlongtap 长按302ms 就会触发但是同时也会触发bindtap事件 如果有tap的话就会触发
    + bindlongpress 长按不会触发tap事件
+ input 事件
    + e.datail.value 获取输入框的值
+ page
    + Page.prototype.setData(object,callback)
+ fouces 默认获取鼠标焦点 
+ chenckbox 单选框事件
    + bindchange 当元素被选中或取消时触发
    + event.detali.value
+ slider 事件
    + chenge 拖动结束之后触发
    + changing 拖动中触发

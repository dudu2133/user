# git基础命令
+ 初始化仓库
    -  git remote add origin https://github.com/dudu2133/yanghao.git
    -  git clone   https://github.com/dudu2133/yanghao.git
+ 添加分支
    - git checkout -b 分支名称  (创建并切换到分支)
    - git checkout 分支名称 (切换分支)
+ 添加最新数据
    - git add . 添加最新的数据
    - git commit -m "最新数据"
    - git push . 上传到远程仓库
